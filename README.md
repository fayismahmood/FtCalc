# FtCalc
This is a Caliculator based on Pyqt5.You can use this after installing Pyqt5 using pip
## STEP 1

**Install pyqt5 using pip**

```sh
pip install PyQt5
```

**run Main.py with python compailer**
```sh
py Main.py
```
